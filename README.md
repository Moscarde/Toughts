## Como usar 🤔

1.  Clone o repositório: `git clone https://github.com/Moscarde/Toughts`
2.  Instale as dependências: `npm install`
3.  Execute o script: `npm start`

## Limitações ⚠️

É necessário configurar o banco de dados mysql no arquivo "/db/conn.js"
